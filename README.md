# Моноплист — Система управления строительным магазином

**Моноплист** — это веб-приложение на **ASP.NET Core 8.0 (Razor Pages)** для автоматизации розничной и мелкооптовой торговли строительными материалами.  
Система включает в себя управление товарами, клиентами, складскими запасами, заказами, поставщиками, отчётами и настройками интерфейса. Поддерживает три языка (русский, английский, казахский), тёмную тему, компактный режим и двухфакторную аутентификацию.

---

## 🚀 Основные возможности

- 📦 **Каталог товаров** – добавление, редактирование, удаление, поиск, фильтрация по категориям, загрузка изображений.
- 👥 **Клиентская база** – управление клиентами с возможностью просмотра паролей (для администратора), сортировка, поиск.
- 🛒 **Продажи / Заказы** – оформление заказов, учет продаж.
- 🏭 **Склад** – контроль остатков, единицы измерения.
- 🚚 **Поставщики** – управление информацией о поставщиках.
- 📊 **Отчёты** – формирование отчётов по продажам, остаткам и т.д.
- 🔐 **Аутентификация и авторизация**
  - Вход сотрудников по логину/паролю или одноразовому ключу восстановления.
  - Вход клиентов по email/паролю с собственной схемой аутентификации.
  - Гостевой режим для просмотра товаров.
  - Роли: **Admin**, **Manager**, **Seller**, **Customer**, **Guest**.
  - Двухфакторная аутентификация (TOTP) для сотрудников.
- 🎨 **Персонализация интерфейса**
  - Пользовательские настройки: язык, тема (светлая/тёмная), компактный режим, цвет акцента.
  - Переключение темы без перезагрузки.
- 🔔 **Центр уведомлений** – просмотр и чтение уведомлений в реальном времени.
- 🌐 **Мультиязычность** – русский, английский, казахский языки интерфейса.
- 📱 **Адаптивный дизайн** – корректное отображение на мобильных устройствах.

## ⚙️ Установка и запуск

### Требования
- [.NET 9.0 SDK](https://dotnet.microsoft.com/download/dotnet/8.0)
- [SQL Management ]([https://www.postgresql.org/](https://learn.microsoft.com/ru-ru/ssms/install/install)) (версия 22 и выше)

Структура проекта
Monoplist/
├── Pages/
│   ├── Account/           # Страницы аутентификации и восстановления пароля
│   ├── Products/          # Каталог товаров (CRUD)
│   ├── Customers/         # Управление клиентами
│   ├── Orders/            # Заказы
│   ├── Warehouse/         # Склад
│   ├── Suppliers/         # Поставщики
│   ├── Reports/           # Отчёты
│   ├── Settings/          # Настройки профиля, безопасности, внешнего вида, уведомлений
│   └── Shared/            # Общие части (боковая панель, заголовок)
├── Models/                # Entity-модели базы данных
├── Data/                  # Контекст базы данных и миграции
├── ViewModels/            # View-модели для страниц
├── wwwroot/               # Статические файлы (изображения, CSS, JS)
├── appsettings.json       # Конфигурация приложения
└── Monoplist.csproj       # Файл проекта
